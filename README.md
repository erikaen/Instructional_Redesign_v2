# Accounting Pre-Work — Prototype v2

The revised (v2) prototype of the **Accounting Pre-Work** for the Yale School of Management /
CMU METALS, by Rick Antle (William S. Beinecke Professor of Accounting).

This is a brand-new build that supersedes the original prototype
([Prework-Prototype](https://github.com/rick50antle/Prework-Prototype), frozen).

## What's here
- **`index.html`** — the landing page (course orientation + roadmap).
- **Module 1** — complete and interactive, five activities:
  1. `11-getting-organized.html` — Getting Organized
  2. `12-insurance-and-maria.html` — Insurance and Maria
  3. `13-anna-and-the-bank-statement.html` — Anna and the Bank Statement
  4. `14-keeping-reasons.html` — Keeping the Reasons
  5. `15-naming-what-you-built.html` — Naming What You Built
- **Modules 2–4 + Bonus** — placeholders on the roadmap (not yet built).

## Running it
Static site, no build step — React is loaded via CDN. Open `index.html` in a browser, or
serve the folder (e.g. `python3 -m http.server`).
